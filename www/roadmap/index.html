Q1 2023 | January - March

  • Forms Extensibility
    Provide the ability to Plug-in custom controls in Forms.
    Planned -
  • Regional settings
    Enable a workflow to format data tpes such as integer, decimal, and date time, to the selected locale and time zone.
    Planned -
  • User and Group Management APIs
    User and group endpoints available to allow ease of managing programmatically. i.e., Adding users, creating groups, updating groups, etc.
    Planned -
  • Workflow Instance Data Deletion
    Ability to remove workflow instances including the initiation data.
    Planned -
  • Button Control in Forms
    Ability to add a button control onto a form configured to perform an event on click.
    Released -
  • Repeating section in Task Forms
    Use the data and repeating section control that was on a Start form or a Task form in subsequent forms within the workflow.
    Cancelled -
  • Product Naming updates
    With the recent renaming of Nintex Workflow Cloud to Nintex Automation Cloud, references to Nintex Workflow Cloud and NWC will be replaced.
    Planned -

Q2 2023 | April - June

  • Forms permissions
    Provide the ability to set user permissions to access Forms.
    Planned -
  • Tasks - long term delegation
    Ability to delegate tasks to another person for a period.
    Planned -
  • Forms extensibility enhancements
    Expand or improve capabilities for custom controls in Forms.
    Planned -
  • Forms global styling
    Ability to adhere to brand compliance by setting or modifying a style for all forms created.
    Planned -
